CSS Div V/S Span



Div and Span tags are both used as content Wrappers.

The difference between the two is DIV is Block Eement and SPAN is In-line Element.

What that means is the DIV tag takes up the entire width of the screen whereas SPAN conforms to the width of whatever element it contains.
