CSS Float V/S Align



CSS Align

You use align to align text and other items rather it be left, right, centered, or justified. Align DOES NOT remove the item from the document flow

CSS Float

Floats an object to the left or right and REMOVES it from the document flow. (i.e. A thumbnail image with paragraph text flowing around it -- you will usually need to set some margins on the image so it looks right).

Float

In this example, the image will float to the left in the paragraph, and the text in the paragraph will wrap around the image.
